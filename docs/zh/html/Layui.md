# Layui



```
经过仔细研究和慎重权衡，前天在本群发布的有关于「解散 QQ 群并将技术反馈转移至 layim 主题专区」之公告，决定如下：

1. 为了这六年多沉淀下来的情感，决定不解散本群。
2. 同时也为了进一步优化工作流程，还望大家 layim 相关的问题统一反馈到主题专区（https://dev.layuion.com/groups/list/layim/），而本群将更名为：layui 主题开发交流，同时为了避免广告滋生，也欢迎活跃者申请群管，共同维护网络安全环境。
3. Layui 的最新动态统一在 Github 和 Gitee 平台获取。 

Layui 经历了许许多多的艰难困苦，如今依然继续探索，破旧立新，作为开源创作者，我由衷地呼吁大家能通过平台化的技术反馈，来减少不必要的消耗，让相对更高效的工作流程得以实现。

—— 贤心 2022.06.08
```
